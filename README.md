# 🌙 Dark Mode Toggle

مشروع بسيط باستخدام JavaScript و CSS لتفعيل الوضع الليلي (Dark Mode).

---

## 🚀 فكرة المشروع

زر واحد يقوم بـ:

- تفعيل الوضع الليلي
- الرجوع للوضع العادي

---

## 🖼️ صورة من المشروع

![Project Screenshot](screenshot.png)

---

## 🛠️ التقنيات المستخدمة

- HTML5
- CSS3
- JavaScript

---

## 🎯 ماذا تعلمت من المشروع؟

- استخدام `classList.add()` و `classList.remove()`
- الفرق بين استخدام `style` و `class`
- مفهوم Toggle
- أهمية ربط CSS مع HTML
- التحكم في شكل الصفحة باستخدام JavaScript
